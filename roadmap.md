## 📊 Daily Practice Guidelines

| Activity | Time | Notes |
|----------|------|-------|
| Learn new topic | 1 hour | Read theory + code examples |
| Solve problems | 2 hours | Start easy, move to medium |
| Review previous | 30 min | Revisit yesterday's problems |
| **Total** | **3.5 hours/day** | Consistency > long sessions |

## 🎯 Interview Readiness Checklist

- [ ] Can solve 2 medium problems in 45 minutes
- [ ] Know all 10 problem-solving patterns by heart
- [ ] Can explain time/space complexity with edge cases
- [ ] Comfortable with OOPs concepts and design
- [ ] Know modern C++ (11/17) and Java (8/17) features
- [ ] Have practiced with the edge case checklist
- [ ] Have completed at least 3 mini projects
- [ ] Can write clean, commented, production-quality code
