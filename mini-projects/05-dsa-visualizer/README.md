# 📊 DSA Visualizer (Console)

## Problem Statement
Interactive console tool that visually demonstrates how common data structures and algorithms work step-by-step.

## Features
- **Sorting Visualizer**: Watch Bubble, Selection, Insertion, Merge, Quick sort step-by-step
- **Binary Search Visualizer**: See the search space narrow with each step
- **Stack/Queue Visualizer**: Push/pop/enqueue/dequeue with visual state
- **BST Visualizer**: Insert, Search, Delete with tree printing
- **Graph Traversal**: BFS/DFS with visited-node highlighting

## Key Learning Value
Seeing algorithms work visually builds far deeper intuition than just reading code.

## DSA Concepts
| Visualization | Algorithm | Key Insight |
|--------------|-----------|-------------|
| Sorting | Bubble, Quick, Merge | Compare swap counts, see why O(n²) vs O(n log n) |
| Binary Search | BS on sorted array | Watch search space halve each step |
| BST | Insert/Search/Delete | Understand tree balance and traversal |
| Graph | BFS/DFS | See level-by-level vs depth-first exploration |
