# 📂 Console File Database

## Problem Statement
Build a simple key-value database that stores records using hashing, supports CRUD operations, and persists data to files.

## Features
- Insert, Get, Update, Delete records
- Hash-based O(1) average lookup
- Collision handling (chaining)
- File persistence (save/load)
- Display with formatted tables

## DSA Concepts
| Concept | Application |
|---------|------------|
| **Hash Table** | Core storage with custom hash function |
| **Chaining** | Collision resolution with linked lists |
| **File I/O** | Persistence layer |
| **Serialization** | Convert records to/from string format |
